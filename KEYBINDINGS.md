# WezTerm 快捷键说明

> 注意：`Alt` 在配置中代替 `Super/Win` 键，避免与 Windows 系统快捷键冲突

## 功能键

| 快捷键 | 功能 |
|--------|------|
| `F1` | 进入复制模式 |
| `F2` | 打开命令面板 |
| `F3` | 打开启动器 |
| `F4` | 标签页模糊搜索 |
| `F5` | 工作区模糊搜索 |
| `F6` | 快捷命令菜单（代理/Agent更新） |
| `F11` | 全屏切换 |
| `F12` | 显示调试信息 |

## 复制粘贴

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+C` | 复制到剪贴板 |
| `Ctrl+Shift+V` | 从剪贴板粘贴 |
| `Shift+Insert` | 粘贴 |
| `鼠标右键` | 复制选中文本 |

## 标签页管理

| 快捷键 | 功能 |
|--------|------|
| `Alt+Enter` | 新建标签页 |
| `Alt+Ctrl+T` | 新建标签页 |
| `Alt+W` | 关闭当前标签页 |
| `Alt+H` | 切换到左侧标签页 |
| `Alt+L` | 切换到右侧标签页 |
| `Alt+Ctrl+H` | 将当前标签页左移 |
| `Alt+Ctrl+L` | 将当前标签页右移 |

|--------|------|

| `Alt+Ctrl+\` | 水平分割（左右分割） |

### 窗格操作
| 快捷键 | 功能 |
|--------|------|
| `Alt+Enter` | 最大化/还原当前窗格 |
| `Alt+W` | 关闭当前窗格 |

### 窗格导航

| `Alt+Ctrl+J` | 切换到下方窗格 |
| `Alt+Ctrl+H` | 切换到左侧窗格 |
| `Alt+Ctrl+L` | 切换到右侧窗格 |
| `Alt+Ctrl+P` | 窗格选择器（数字键选择） |

## 窗口管理

## Leader 键模式

先按 `Alt+Ctrl+Space` 激活 Leader 模式，然后：

### 字体调整模式（Leader + F）
| 快捷键 | 功能 |
|--------|------|
| `K` | 放大字体 |
| `J` | 缩小字体 |
| `R` | 重置字体大小 |
| `Esc` 或 `Q` | 退出模式 |

### 窗格调整模式（Leader + P）
| 快捷键 | 功能 |
|--------|------|
| `K` | 向上扩展窗格 |
| `J` | 向下扩展窗格 |
| `H` | 向左扩展窗格 |
| `L` | 向右扩展窗格 |
| `Esc` 或 `Q` | 退出模式 |

## 快捷命令（F6）

按 `F6` 打开快捷命令菜单，包含：

1. **Set Proxy (Windows)** - 设置临时代理（Windows PowerShell）
   ```powershell
   $env:http_proxy="http://127.0.0.1:2080"
   $env:https_proxy="http://127.0.0.1:2080"
   ```

2. **Set Proxy (Linux)** - 设置临时代理（Linux/macOS）
   ```bash
   export http_proxy=http://127.0.0.1:2080
   export https_proxy=http://127.0.0.1:2080
   ```

3. **Agent Update** - 更新 AI Agent CLI 工具
   ```bash
   claude update
   opencode
   npm upgrade -g @openai/codex
   npm upgrade -g @google/gemini-cli
   ```

## SSH 连接

SSH 连接已移至域配置（`config/domains.lua`）。

使用方式：
1. 按 `F3` 打开启动器
2. 选择配置的 SSH 服务器
3. 像本地标签页一样使用远程终端
